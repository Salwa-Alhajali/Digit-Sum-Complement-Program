# 🔢 Digit Sum Complement Program

## 📌 Description

This program reads a number as a string, calculates the sum of its digits, and then appends a digit to the original number so that the total sum of digits becomes **10**.

---

## ⚙️ How It Works

* The user enters a number as a string.
* The program calculates the sum of all digits in the number.
* It computes the missing value needed to reach 10.
* If needed, it appends that value to the original number.
* If no digit is needed, the original number is printed as it is.

---

## 💡 Idea Behind the Program

The main idea is to manipulate a number at the string level and adjust its digit sum to a fixed target (10) using simple arithmetic and iteration.

---

## 🚀 Example

**Input:**
123

**Process:**
Sum = 1 + 2 + 3 = 6
Missing = 10 - 6 = 4

**Output:**
1234

---

## 📝 Notes

* Input should consist only of numeric characters.
* The logic assumes the sum is less than or equal to 10.
* For larger sums, the program may need adjustments.

---

## 🔧 Possible Improvements

* Add input validation (ensure digits only).
* Handle cases where sum exceeds 10.
* Extend functionality for different target sums.
